What this site demonstrates in AppDynamics Browser RUM

  • Multi-page user journeys (Experience Journey Map + Sessions)
  • AJAX latency + failures (Pages & AJAX Requests)
  • JavaScript errors (JavaScript Errors)
  • Slow page load (End User Response Time distribution/trend)
  • Resource timing (Resource Performance)
  • SPA / virtual pages (Single Page App tracking)
  • Geo simulation labels (creates “virtual locations” for demo segmentation)

5‑minute demo runbook

  1. Open Journey and click Run Journey (generates realistic flow).
  2. Open Search/AJAX and run Burst (mix of slow + failing calls).
  3. Open JS Errors and trigger Uncaught + Promise rejection.
  4. Open Slow once (creates slow load outliers).
  5. Open Geo‑Sim and click a few “cities” (creates location-tagged virtual pages).

Where to view in AppDynamics

  • User Experience → Browser Apps → rum-demo
  • Pages & AJAX Requests (page load + XHR)
  • JavaScript Errors (error types + stack traces)
  • Experience Journey Map (flows)
  • Sessions (session drill-down)

Tip: While generating traffic, set AppDynamics time range to Last 15 minutes for quick feedback.
